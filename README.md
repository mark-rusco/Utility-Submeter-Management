# Utility Submeter Management PWA

A mobile-first Progressive Web App (PWA) for managing Electricity and Water utility billing using submeter readings.

## Features

- Dashboard with Outstanding Balances, Collection Summary, and My Bill
- Electricity and Water Monthly Bill Management
- Automatic Water Rate Calculation (Charges ÷ Consumption)
- Configurable Electricity Rate (₱/kWh)
- Occupant / Unit Management
- Automatic Previous Reading Lookup
- Payment Tracking (Paid / Unpaid)
- Thermal Receipt Style Printing
- Owner Balance Calculation
- History, Search, and Filters
- Import / Export Backup
- Offline Local Storage
- Mobile-First Responsive Design
- Installable PWA

## My Bill Formula

My Bill = Main Bill - Total Occupant Charges

## Data Storage

All data is stored locally in the browser using Local Storage.

## Backup

Use Export Data regularly to create JSON backups.
